## 📱 About
Silai is a Smart Tailor Booking Android App that helps users 
find nearby tailors and book home pickup & delivery for 
stitching clothes.

**"Smart Tailor at Your Doorstep"**

## ✨ Features

### 👤 User Side
- Register & Login (SQLite)
- Select area to find tailors
- View tailors with ratings & images
- Book home pickup with date & time
- Track order status live
- Upload reference cloth image
- Push notifications

### 🔐 Admin Side
- Admin login (admin/1234)
- View & update all orders
- Add new tailors
- Dashboard with stats

## 🛠️ Tech Stack
| Technology | Purpose |
|---|---|
| Kotlin | Programming language |
| Android Studio | IDE |
| SQLite | Local database |
| Glide | Image loading |
| RecyclerView | List rendering |
| SharedPreferences | Session storage |

## 🚀 How to Run
1. Clone the repo
2. Open in Android Studio
3. Sync Gradle
4. Run on device/emulator
